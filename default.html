Experience Stunning Sliders

dSlider

Full Featured Lightweight Responsive Slider/Carousel with Pure JS

dSlider is a versatile and customizable slider library that empowers you to create captivating image sliders for your website. It offers a wide range of options and configurations to make your content shine.

Simple Slider

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-simple', }); });

Fade Slider

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-fade', sliderStyle: "fade", // default: slide }); });

Simple Carousel

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-carousel', responsiveSlides: { 0: 3 }, // default: 1 gap: 2 // default: 0 }); });

Responsive Carousel

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-responsive', responsiveSlides: { 1200: 3, 800: 2, 0: 1 }, // default: 1 gap: 2 // default: 0 }); });

Responsive Non-Loop Carousel

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-end', responsiveSlides: { 1200: 3, 800: 2, 0: 1 }, // default: 1 infiniteLoop: false, // default: true gap: 2 // default: 0 }); });

Responsive RTL Carousel

document.addEventListener("DOMContentLoaded", function() { dSlider({ sliderID: 'dSlider-rtl', responsiveSlides: { 1200: 3, 800: 2, 0: 1 }, // default: 1 gap: 2, // default: 0 sliderRTL: true // default: false }); });
